# Spark

a 2D LAN free-for-all multiplayer game!

## Installation

currently only works on arch linux but soon should be fixed.  
from the aur install:
1. crystal 
2. shards
3. voidcsfml
  
run `shards install`
and then `crystal build src/APF.cr`
finnaly run it with `./APF`
